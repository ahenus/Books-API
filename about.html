Főoldal Könyvek

About

Ez a projekt egy statikus, JSON-alapú digitális könyvtár, amely REST-szerű API struktúrát modellez kliensoldali megvalósítással.

Az adatok strukturált JSON fájlokban találhatók, amelyeket JavaScript tölt be dinamikusan. A rendszer kliensoldali paginációt alkalmaz.

Technikai megvalósítás

  • HTML5 + CSS3
  • Modern JavaScript (ES6)
  • Statikus JSON endpointok
  • GitHub Pages deploy
